# AI Chat Assistant Web Application

A web-based AI chat assistant built using JSP, Servlets, and JDBC that integrates the Gemini API to generate responses to user queries.

## Tech Stack
- Java
- JSP
- Servlets
- JDBC
- MySQL
- Google Gemini API

## Features
- Real-time AI response generation
- Server-side session management for chat context
- Chat history storage in MySQL

## Status
Initial version. Improvements and deployment coming soon.
